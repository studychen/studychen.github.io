域名+key 获取新的 url

还记得刚才的代码里我们给图片指定了 key 吗，只是图片在七牛空间里的唯一表示。

刚才我们指定的 key 为testimage，看新的图片 url 为http://7xq7ik.com1.z0.glb.clouddn.com/testimage，新url就是刚才的 key 结尾的。

前面一大串的字符是七牛给我们的默认域名，你也可以自定义域名。
